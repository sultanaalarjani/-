# لوحة إدارة المهام (Task Dashboard)

لوحة تحكم لإدارة المهام والمشاريع مبنية باستخدام **React + Vite**، بتصميم عربي (RTL) وواجهة داكنة عصرية.

## ✨ المميزات

- **لوحة Kanban** بثلاثة أعمدة: قيد الانتظار، جاري التنفيذ، مكتملة.
- **السحب والإفلات** (Drag & Drop) لنقل المهام بين الأعمدة.
- **بطاقات إحصائيات** لعرض إجمالي المهام وحالتها.
- **شريط تقدّم** يوضح نسبة الإنجاز الكلية.
- **إضافة مهام** مع عنوان، وصف، أولوية، حالة، وتاريخ استحقاق.
- **حفظ تلقائي** في المتصفح (localStorage) — بياناتك تبقى محفوظة.
- **تصميم متجاوب** يعمل على الجوال والحاسب.

## 🚀 التشغيل

```bash
# تثبيت الحزم
npm install

# تشغيل وضع التطوير
npm run dev

# بناء نسخة الإنتاج
npm run build

# معاينة نسخة الإنتاج
npm run preview
```

ثم افتح الرابط الظاهر في الطرفية (عادةً `http://localhost:5173`).

## 🗂️ هيكل المشروع

```
src/
├── App.jsx                # المكوّن الرئيسي وإدارة الحالة والحفظ
├── main.jsx               # نقطة الدخول
├── index.css              # كل التنسيقات
├── data/
│   └── seed.js            # الأعمدة والمهام الافتراضية
└── components/
    ├── Sidebar.jsx        # القائمة الجانبية
    ├── StatsCards.jsx     # بطاقات الإحصائيات
    ├── Board.jsx          # لوحة Kanban
    ├── TaskCard.jsx       # بطاقة المهمة
    └── TaskModal.jsx      # نافذة إضافة مهمة
```
